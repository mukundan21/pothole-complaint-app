# 🚧 Pothole Detector iOS App

An AI-powered iOS application developed using Swift and CoreData that allows users to detect, store, and manage pothole reports efficiently.

## 📱 Features

- Capture pothole details
- Store data using CoreData
- Persistent local storage
- Clean SwiftUI interface
- Structured data management

## 🛠️ Tech Stack

- Swift
- SwiftUI
- CoreData
- Xcode

## 🗂️ Project Structure

- ContentView.swift – Main UI logic
- Persistence.swift – CoreData configuration
- PotholedetectorApp.swift – App entry point
- Potholedetector.xcdatamodeld – Data model

## 🎯 Purpose

This project is part of my AI-based road anomaly detection system.  
Phase 1: Pothole Detection (Model)  
Phase 2: iOS App Integration
